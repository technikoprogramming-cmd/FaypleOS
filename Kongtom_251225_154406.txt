## ðŸ›  Setup Guide: Nikodem's OS Frontend

This is a **Frontend Template**. It provides the glassmorphism UI, terminal, and dashboard, but you must "plug in" your own server details in the code to make the buttons and stats work.

### 1. Identify the OS Name

If you are sharing this, ensure people know it is the **Nikodem's OS Frontend**.

* **In the Code:** * Look for `<title>FaypleOS | Server Template</title>`
* You can change this to `<title>Nikodem's OS Frontend</title>`



### 2. Setting the Connection (IP & Port)

To see your server address on the dashboard and in the system info:

* **Dashboard Stats:** Find the line `<div class="stat-val">IP: HERE GOSE IP:PORT</div>`. Replace `HERE GOSE IP:PORT` with your actual address (e.g., `192.168.1.50:8080`).
* **The "Link" Button:** Find the button that opens the file manager:
* `onclick="window.open('HERE PUT http ip with port...')"`
* **Change to:** `onclick="window.open('http://your-server-ip:port')"`



### 3. User & Hardware Identity

The frontend needs to know who is "logged in" for the display:

* **Username:** Find `HERE PUT YOUR USERNAME`. Change it to your server's root or admin name. This makes the terminal look like a real Debian session.
* **Host Name:** Find `HERE PUT SERVER NAME`. Change it to the name of your physical machine (e.g., "Nikodem-Main-Server").

### 4. How to Edit the Code

1. **Right-click** your `.html` file and select **"Open with..."** and choose **Notepad** or **VS Code**.
2. Use **Ctrl+F** (Find) to search for the keywords: `HERE PUT`, `IP:PORT`, or `USERNAME`.
3. Replace the text inside the quotes or tags with your data.
4. **Save** and refresh your browser.
